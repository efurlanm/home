    Propriedades e Mtodos de Classe: Classes podem ter propriedades e mtodos que pertencem prpria classe e no s instncias individuais. Eles so definidos usando a palavra-chave @classmethod ou @staticmethod.

    class MinhaClasse:
    contador = 0  # Propriedade de classe

    @classmethod
    def incrementar_contador(cls):
        cls.contador += 1

    @staticmethod
    def metodo_estatico():
        return "Este  um mtodo esttico."

    Encapsulamento e Modificadores de Acesso: Em Python, voc pode sugerir o encapsulamento de atributos com um nico sublinhado _ para indicar que so protegidos, ou com dois sublinhados __ para indicar que so privados. Isso no impede o acesso, mas uma conveno para sinalizar que esses atributos no devem ser acessados diretamente.

    class MinhaClasse:
    def __init__(self, valor):
        self._protegido = valor  # Indica que deve ser tratado como protegido
        self.__privado = valor  # Indica que deve ser tratado como privado

    Propriedades (Properties): Em vez de acessar diretamente os atributos de uma classe, voc pode usar propriedades para controlar o acesso a eles. Isso feito usando os decoradores @property.

    class MinhaClasse:
    def __init__(self, valor):
        self._valor = valor

    @property
    def valor(self):
        return self._valor

    @valor.setter
    def valor(self, valor):
        if valor < 0:
            raise ValueError("Valor deve ser positivo")
        self._valor = valor

    Mdulos e Pacotes: Classes podem ser organizadas em mdulos (arquivos .py) e pacotes (diretrios com um arquivo init.py). Isso ajuda a estruturar o cdigo de forma modular e reutilizvel.

    Sobrescrita de Mtodos (Override): Classes podem sobrescrever mtodos herdados para modificar ou estender seu comportamento. Isso feito simplesmente redefinindo o mtodo na subclasse.

    class Animal:
    def som(self):
        return "Algum som"

class Cachorro(Animal):
    def som(self):
        return "Latido"

    Mtodos Especiais: Existem outros mtodos especiais alm do init, como str, repr, len, getitem, e muitos mais, que permitem definir comportamentos especficos para objetos da classe.
